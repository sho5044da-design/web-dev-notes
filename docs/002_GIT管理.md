
# GitHubに学習内容を保存する手順

## 目的

作ったドキュメントをGitHubに保存する。

---

## ① ターミナルを開く

Macで

Command + Space

↓

terminal

↓

Enter

---

## ② 教科書フォルダへ移動

```bash
cd ~/web-dev-notes
```

確認

```bash
pwd
```

結果

```text
/Users/shogo/web-dev-notes
```

ならOK。

---

## ③ 新しいドキュメントを作る

例

```bash
touch docs/003_ターミナル基礎.md
```

またはFinderで作ってもよい。

---

## ④ ドキュメントを編集

Finder

↓

web-dev-notes

↓

docs

↓

003_ターミナル基礎.md

↓

内容を書く

↓

保存

---

## ⑤ Gitに変更を認識させる

```bash
git status
```

実行。

例

```text
Untracked files:
docs/003_ターミナル基礎.md
```

意味

「新しいファイルを発見した」

---

## ⑥ 管理対象に追加

```bash
git add .
```

実行。

もう一度

```bash
git status
```

実行。

例

```text
Changes to be committed:
new file: docs/003_ターミナル基礎.md
```

意味

「次の保存対象になった」

---

## ⑦ 履歴として保存

```bash
git commit -m "ターミナル基礎を追加"
```

意味

「今の状態を記録」

---

## ⑧ GitHubへアップロード

```bash
git push
```

意味

「PCの履歴をGitHubへ送る」

---

## 毎回やること

ドキュメント編集

↓

```bash
git add .
```

↓

```bash
git commit -m "変更内容"
```

↓

```bash
git push
```

以上。
